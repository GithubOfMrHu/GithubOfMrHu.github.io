Linear transformation 线性变换

Matrices as Linear transformations

Unfortunately, no one can be told what the Matrix is. You have to see it for yourself.

transformation

  • Transformation is essentially a fancy word for “function”. Well, it’s to be suggestive of a certain way–using movement to visualize this input-output relation.
  • Vector input and Vector output

    满足以下两个性质：

  • lines remain lines
    直线依旧是直线
  • Origin remains fixed
    原点保持固定
    In general, keeping grid lines parallel and evenly spaced
    即网格线保持平行且等距分布。

    How to describe a linear transformation numerically?

    • matrix
    • $\widehat{i}=\begin{bmatrix}0\0\end{bmatrix}\widehat{j}=\begin{bmatrix}0\0\end{bmatrix}\rightarrow\widehat{i}=\begin{bmatrix}1\-2\end{bmatrix}\widehat{j}=\begin{bmatrix}3\0\end{bmatrix}$
    • $\begin{bmatrix}x\y\end{bmatrix}=x\begin{bmatrix}1\-2\end{bmatrix}+y\begin{bmatrix}3\0\end{bmatrix} =\begin{bmatrix}x+3y\-2x+0y\end{bmatrix}$

      2x2Matrix

      线性变换的一种表达方式

      视角

      矩阵是线性变换的一种表达方式，而线性变换可以看做对空间的一种特定变换。
