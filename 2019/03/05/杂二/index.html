month_between函数

month_between函数返回两个日期之间的月份数。如果两个日期月份内天数相同，或者都是某个月的最后一天，返回一个整数，否则，返回数值带小数，以每天1/31月来计算月中剩余天数。

使用month_between等处理日期的函数，要注意参数的数据格式，如数据库中的数据为1937-02-19，要用to_date(birthday,’yyyy-mm-dd’)转换成与sysdate相同的数据格式。

TRUNC函数

  • 处理number型数字。
    TRUNC(number[,decimals])
    number 待做截取处理的数值；decimals 指明需保留小数点后面的位数，可选项，忽略它则截去所有的小数部分,并且截取时并不对数据进行四舍五入。
  • 处理日期。
    TRUNC(date[,fmt])
    date 一个日期数据；fmt 日期格式。该日期将按指定的日期格式截取；忽略它则由最近的日期截取。如下：

floor、ceil、round函数

  • floor(x) 向下取整，与trunc都可以实现截去小数部分。
  • ceil(x) 向上取整。
  • round(x) 四舍五入。

LeetCode Q3 Longest Substring Without Repeating Characters

Given a string, find the length of the longest substring without repeating characters.

THINKING

  • 首先考虑为空和字符串s仅包含一个字符的情况，直接返回s本身。
  • 之后考虑复杂情况，将s赋值给字符数组str，通过head和tail记录不重复子字符串的起始和终止字符在str中的下标(之后发现并不需要记录tail)，len记录当前不重复子字符串的最大长度。
    • 从前至后遍历str，当前下标为i的元素通过遍历和head和tail之间的元素进行比较：
      • 若没有相同的字符 则 tail++;
      • 若有相同的字符，即str[i]==str[j],
        则head=j+1;tail++;
        当len<tail-head+1;时，len=tail-len+1;
    • 返回len
