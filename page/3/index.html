Hexo个性化设置

发表于 2019-03-22 | 更新于 2020-04-15 | 课程 Hexo | 阅读次数：

如何使Hexo Next主题支持Latex数学公式

过程

由于我使用的Next主题,在路径为\blog_path\themes\next\docs\MATH.md的文档中找到了官方给出的支持数学公式的方法：需要修改主题的配置文件打开数学公式渲染引擎，再使用对应的Hexo渲染引擎即可。但是根据此文档修改后遇到了以下问题：

  1. 执行npm un和npm i命令更换Hexo渲染引擎后某些标签无法渲染，如<!-- more -->无法正常截断预览内容。
  2. 在使用hexo-renderer-pandoc渲染引擎后，执行hexo -g出现报错。

解决方案

之后寻找如何在不更换渲染引擎的前提下实现渲染。发现可以直接在blog_path路径下执行npm install hexo-renderer-mathjax --save命令，再在需要渲染的post的声明中通过mathjax: true打开渲染即可。默认情况下关闭，提高网页的加载速度。
