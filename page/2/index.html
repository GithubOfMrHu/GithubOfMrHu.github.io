    书单

  • 《禅与摩托车的维修艺术》
  • The 4-Hour Work Week 《每周工作四小时》

    大纲

  1. 扎克伯格与他2009至今每年的新年计划。
  2. 阿斯伯格综合征（AS）与情绪的作用，心理咨询的作用很多时候在于帮助咨询者理解和接受那个时候为什么要那么做，并没有一定“正确”的解决方式。
  3. 计划是否要告诉别人。告诉别人可以带来一些外在压力和动力，但也可能因为告诉别人获得的满足感而降低完成计划的动力。
  4. 相比具体的目标，把日常可以完成的动作作为计划的内容，可以避免在计划的后期因为完不成目标产生挫败感，和因此带来的自暴自弃，更容易养成良好的习惯，做更多的事。
  5. 意志力类似肌肉，可以通过锻炼增强自己的意志力，但消耗意志力的行为和计划依然是难以长久坚持下去的，尝试去改变环境和自身来使它变成一件开心的事。
  6. 把会带来心理压力的大目标进行分解，分解成可执行的，感受上不会费力的小目标。
  7. 环境给人的影响作用是巨大的，把自己的身体当成实验工场，试验在什么环境和条件下会提高工作效率，有比较好的状态。
  8. 计划本身不重要，做计划很重要。当做好计划后，想到时大脑就会对计划中涉及到的区域进行预热。
  9. 习惯的力量。核心习惯的养成会带动其它的习惯。
  10. 如何选择创业的方向，别人觉得费力的，而自己觉得是在玩的事情。
  11. Just do it.

如何迁移hexo博客至另一台主机（windows操作系统)

由A主机迁移至另一台主机B

准备

  1. 在新PC上安装下列应用程序：
  • Node.js (Should be at least nodejs 6.9)
  • Git

    拷贝

  1. 拷贝主机A中的blog文件夹至主机B的任意目录,如D:\blog。

  2. 在主机A控制台执行npm config ls命令，查看

    path下的node_modules是node.js全局安装的路径，主机A路径是
    C:\Users\user\AppData\Roaming\npm\node_modules，拷贝此路径下的hexo-cli文件夹至主机B对应目录下。

  3. 若使用VS Code作为IDE，想继续使用相关插件，则需要拷贝主机AMicrosoft VS Code install path的\resources\app路径下的extensions文件夹至主机B VS Code对应目录下。

    修改主机B的环境变量

  4. 在系统变量中的Path变量中添加npm的安装路径C:\Users\user\AppData\Roaming\npm。
  5. 在系统变量中的Path变量中添加node.js的安装路径。
  6. 在blog文件夹下\node_modules\.bin目录下有Hexo的windows系统脚本文件，在系统变量中的Path中添加此路径：D:\blog\node_modules\.bin。

lambda

什么是lambda

“Lambda 表达式”(lambda expression)是一个匿名函数，Lambda表达式基于数学中的λ演算得名，直接对应于其中的lambda抽象(lambda abstraction)，是一个匿名函数，即没有函数名的函数。Lambda表达式可以表示闭包（和数学领域中lambda不同）。

使用lambda可以使代码更简洁。

lambda in java 8

A simple example

特性

  • 可选类型声明：不需要声明参数类型，编译器可以统一识别参数值。
  • 可选的参数圆括号：一个参数无需定义圆括号，但多个参数需要定义圆括号。
  • 可选的大括号：如果主体包含了一个语句，就不需要使用大括号。
  • 可选的返回关键字：如果主体只有一个表达式返回值则编译器会自动返回值，大括号需要指定明表达式返回了一个数值。

lambda表达式只能引用标记了final的外层局部变量，即不能在lambda内部修改定义在域外的局部变量，否则会编译错误。
